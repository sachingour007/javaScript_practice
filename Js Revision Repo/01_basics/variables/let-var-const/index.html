Variable in Details

Basically 3 type of variables are available in JS

  • Var => is a Function scope variable. we can redecalare the var and reinitilized, var can b hoisted.
  • Let => is a Block Scope Variable. only can reinitilized the variable, is also hosited but these created inside the script so these js engine not access the varialble, give the reference Error: not access before initilization.
  • Const => is Block Scope Variable. not reinitilized and redeclared the variable, is also hosited but these created inside the script so these js engine not access the varialble, give the reference Error.
